# movies-explorer-frontend
--
Верстка макета дипломной работы в рамках Яндекс Практикума
--

Добавлен следующий функционал:
 - Авторизиция;
 - Регистрация;
 - Поиск фильмов на руссокои и английском языке;
 - запись фильмов в личную библиотеку
